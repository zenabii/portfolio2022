CSM

see website

Community Science Museum was my first semester project at Noroff. This is a basic html and css website with no JavaScript at all. It was completely open design wise, but it did require specific landing pages and context in the project.

The target group in this project was children, but it's not easy to engage children through a museum project. So I knew I wanted to add some untraditional graphics to the historic statues and monuments to give it more life and perhaps appeal more to kids.

The colour palette in this project is very straight forward, playing around with neutral, warm tones and a very clear yellow contrast. This yellow worked great to emphasize more important content and also great colour contrast to the additional graphis on the images.

One of several attempts to appeal more to kids was adding VR goggles to one of the statues. It has been proven several times that we have to think more creative if we want to teach kids something new or historic. I believe a true VR experience at a museum would appeal to many kids, and even adults.
